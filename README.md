# Portfolio Site

Prosty projekt Next.js + React + Tailwind z trzema sekcjami:

1. Baner (`#baner`) — pełny ekran z menu
2. Portfolio (`#portfolio`) — przykładowe karty
3. Kontakt (`#kontakt`) — formularz kontaktowy

## Jak uruchomić

1. Zainstaluj Node.js (wersja 18 lub nowsza).
2. W katalogu projektu uruchom:

```bash
npm install
npm run dev
```

3. Otwórz `http://localhost:3000` w przeglądarce.

## Struktura

- `app/page.tsx` — strona główna z trzema sekcjami
- `app/globals.css` — style globalne Tailwind
- `tailwind.config.ts` — konfiguracja Tailwind
- `package.json` — skrypty i zależności
